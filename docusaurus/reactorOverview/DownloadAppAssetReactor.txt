Reactor Name
DownloadAppAssetReactor

Description
Download a file or directory from the project's assets folder. If a directory is specified, it will be zipped before download.

Required keys
PROJECT, FILE_PATH

Key description
PROJECT: The unique id for the project/app
FILE_PATH: The relative path of the file or directory to download (do not include '/version/assets/' prefix)

Optional keys
None

Usage
DownloadAppAsset(project="myProjectId", file_path="myfile.txt")
DownloadAppAsset("myProjectId", "myfile.txt")
