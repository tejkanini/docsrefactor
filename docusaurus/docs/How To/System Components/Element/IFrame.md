# IFrame

You can embed a web page into your App interface using the IFrame block. Drag and drop the IFrame block into your app UI, as highlighted below.

![IFrame](../../../../static/img/Systemcomponents/Iframe.png)

## Block Settings

### Content

Specify the link of the webpage you want to embed and give it a title.

### Appearance

Customize the frame by setting its height and width, margin (space around the element), padding (space inside the element), and border size, color, and style.

Congratulations, you have successfully added an IFrame block to your UI.
